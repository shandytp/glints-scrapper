# Glints Scrapper

### Goals nya
---
1. Input:
    - Cari nama Job: DA/DS/DE
    - Lokasi
2. List job, lokasi, gaji, skill, deskripsi
3. Bisa bikin ETL & Streamlit

### How to Run

Untuk melakukan query jobs, bisa menggunakan command

```bash
python main.py --jobs "insert_jobs_name" --filename "filename.json"
```
